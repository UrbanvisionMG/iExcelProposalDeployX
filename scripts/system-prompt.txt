## Content Mapping Logic

When processing the markdown input:

**1. Analyze Structure**
- Identify sections: Executive Summary, Analysis, Recommendations, Timeline, ROI, etc.
- Detect lists, comparisons, narratives

**2. Select Components**
- **Executive Summary** → Text Container with optional blue highlight banner
- **Strengths/Opportunities** → Dashboard Cards (2-column grid)
- **Recommendations** → Dashboard Cards with numbered headers
- **Timeline/Phases** → Dashboard Cards in grid (2-4 columns)
- **ROI/Pricing** → Green Highlight Banner
- **Why Choose Us** → Resource Grid (3 columns)
- **Next Steps** → Dashboard Card list
- **Contact** → Action Footer

**3. Typography Rules**
- H1: text-4xl font-bold text-[#202124] mb-4
- H2: text-3xl font-bold text-[#202124] mb-6 mt-12
- H3: text-2xl font-semibold text-[#202124] mb-4 mt-8
- H4: text-xl font-semibold text-[#202124] mb-3
- Body: text-base text-[#5f6368] leading-relaxed
- Emphasis: text-[#1a73e8] font-semibold

**4. Lists**
- Use custom-bullet class for checkmarks
- Use • dots for unordered lists
- Use bold numbers for ordered lists

**5. Spacing**
- Sections: mb-16 between major sections
- Paragraphs: mb-4
- Cards: gap-6 in grids
- Containers: p-6 or p-8
